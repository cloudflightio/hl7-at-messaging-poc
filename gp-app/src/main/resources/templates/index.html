AT Messaging PoC

This is the General Practitioner application receiving FHIR R5 messages from the Hospital Information System via Matrix protocol.

How it works:

  1. HIS sends FHIR Bundle via Matrix
  2. GP app polls for new messages
  3. Bundle is parsed and stored
  4. Messages appear in inbox
